-----------------------------------------------
v1.0 Beta  —  2026-06-17
-----------------------------------------------

初始发布：
  • 文件夹监控，每15秒轮询检查。
  • 按AI工具自动分类文件：
    ChatGPT、Gemini、Tensor Art、Kling、Higgsfield。
  • 自动移动到 Profiles/<配置文件名>/<YYYY-MM>/images|videos/。
  • 每日生成次数扣除（ChatGPT：8次，Gemini：20次）。
  • Tensor Art 每月信用额度扣除（50.03/月）：
    - 图片：每次0.2信用点。
    - 视频：根据时长和分辨率（480p/720p）计算，
      使用 OpenCV 读取文件元数据。
  • Kling（1个视频）和 Higgsfield（1个视频）的月度计数器。
  • 每月17日重置周期。
  • 移动记录保存在 history.txt 中。
  • 使用 plyer 的桌面通知。
  • tkinter 图形界面，包含图片库和视频列表。
  • 每个配置文件的信用与生成次数表。
  • 动态配置文件选项卡（从界面添加/删除）。
  • .bat 启动器，快速访问。
  • 多语言支持（英语、西班牙语、中文）。
  • 深色/浅色主题支持。
  • GitHub 自动更新程序（update.bat）。
